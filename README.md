# Monitoring

Модуль для вывода информации о серверах. Поддерживает вывод дополнительной информации о серверах (требуется плагин - https://github.com/Pisex/cs2-PlayersListCommand)

**Для работы статистики необходимо включить CRON режим. Подробнее в документации** - https://docs.flute-cms.com


Module for displaying information about servers. Supports displaying additional information about servers (requires a plugin - https://github.com/Pisex/cs2-PlayersListCommand)

**For statistics to work, you need to enable the CRON mode. More in the documentation** - https://docs.flute-cms.com




## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **2.0.4**
